 ## How to Run on local
  - Clone the repository
  ##### For POC 1: 
   - Checkout to the **poc-1** tag
   - Run `yarn`
   - Run `yarn storybook`
  ##### For POC 2:
   - Checkout to **master** branch
   - Run `yarn`
   - Run `yarn storybook`
